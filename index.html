Learn Mysql

1) check the mysql version

Syntax: mysql  --version

2) Connect mysql with the databases

Syntax: mysql   -u   root   -p  -h   localhost

where -u: user name, -p: password, -h: host name

3) For showing the list of the databases

Syntax: show databases;

4) For Creating new database

Syntax: create database nameofdatabase;

5) For Selecting the database

Syntax: use databasename;

6) For showing(know) which database is curently selected

Syntax: select database();
