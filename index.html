1.2) What is DEFAULT how to use it

when we are inserting(adding) the data(records) into to column(feilds).
at that time if we do not add any feild value then by default which value will store in that column(feild).
for that we use default value.
Default value will store in database when we do not insert any fild value at the inserting other feild value time.

Syntax: create table tablename
(
column_name   data_type   not null   default value,
column_name   data_type   not null   default value
);

Example:  create table students
(
name   varchar(50)   not null   default "unnamed",
class   int   not null   default 1,
roll_no   int   not null,
school_name   varchar(100)   not null   default "janakpur school"
);

1.3) What is primary key in table. how to use

The PRIMARY KEY constraint uniquely identifies each record in a table.
Primary key must contain UNIQUE values, and can not contain NULL Values.

Syntax: create table tablename
(
column_name   data_type   not null   default value,
column_name   data_type   not null   default value primary key(column_name)
);

Example:  create table students
(
id   int   not null,
name   varchar(50)   not null   default "unnamed",
class   int   not null   default 1,
roll_no   int   not null,
school_name   varchar(100)   not null   default "janakpur school" primary key(id)
);

1.4)What is AUTO_INCREMENT in Table and how to use

Auto Increment allows a unique number to be generated automatically when a new record(data) is inserted into a table

Syntax: create table tablename
(
column_name   data_type   not null   default value auto_increment,
column_name   data_type   not null   default value primary key(column_name)
);

Example:  create table students
(
id   int   not null auto_increment,
name   varchar(50)   not null   default "unnamed",
class   int   not null   default 1,
roll_no   int   not null,
school_name   varchar(100)   not null   default "janakpur school" primary key(id)
);

SQL Aggregate Functions

  • COUNT( )
    • this function returns the number if row that matches a specified criteria.
    1) Syntax: select count(age) from students;
    2) Syntax: select count(id) as students from students;
  • MAX( )
    • Syntax: select max(age) from students;
  • MIN( )
    • Syntax: select min(age) from students;
  • AVG( )
    • This function returns the average value of a numeric column.
    Syntax: select avg(age) from students;
  • SUM( )
    • This function returns the total sum of a numric column.
    Syntax: select sum(age) from students;
